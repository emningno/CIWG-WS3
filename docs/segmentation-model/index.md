# Segmentation Model

{: .draft }
> This deliverable is under active development.

## Purpose

This deliverable defines a segmentation model for Peppol network participants 
with respect to trust and security requirements. The model provides a structured 
basis for applying differentiated trust policies, credential requirements, and 
access controls to distinct participant categories.

It feeds directly into the [Future Trust Architecture](../future-trust-architecture/) 
and informs the [Peppol-ERDS Gap Analysis](../erds-gap-analysis/).
