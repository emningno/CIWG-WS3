# Peppol-ERDS Formal Gap Analysis

{: .draft }
> This deliverable is under active development.

## Purpose

This deliverable provides a formal, requirement-by-requirement assessment of 
the gaps between the current Peppol trust model and the requirements for 
Electronic Registered Delivery Services (ERDS) under eIDAS 2.0 
(Regulation (EU) 2024/1183).

It establishes the evidential basis for WS3 policy positions on ERDS 
applicability and any proportionality arguments advanced to regulators or 
DG GROW.
