# Future Trust Architecture

{: .draft }
> This deliverable is under active development.

## Purpose

This deliverable describes the target trust architecture for the Peppol network, 
taking into account regulatory drivers (eIDAS 2.0, ViDA, EUBW), emerging 
identity technologies, and the operational constraints of the existing 
four-corner model.

It is the primary forward-looking output of WS3 and serves as the architectural 
foundation for the [Evolution Path Assessment](../evolution-path-assessment/).
